# steps
Set ~/.bashsrc variables

    Open a terminal - CTRL+Alt+T.
    Edit file ~/.bashsrc by

nano ~/.bashsrc

    Add the variable to the bottom of the file:
        export AOC_SESSION=________________________________________________
    Save and close nano.
    Apply changes by:

source ~/.bashsrc
